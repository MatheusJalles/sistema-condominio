# 🏢 Sistema de Gestão de Condomínios

Aplicação web completa para gerenciamento de condomínios — controle de moradores, reservas de espaços comuns e comunicados internos.

> **Status:** ✅ Online — hospedado no Firebase

---

## 🎯 Problema que resolve

Condomínios geralmente dependem de planilhas ou grupos de WhatsApp para organizar moradores e reservas. Este sistema centraliza essas funções em uma interface web simples e acessível para administradores e moradores.

---

## 🚀 Funcionalidades

- ✅ Cadastro e controle de moradores
- ✅ Reserva de espaços comuns (salão, churrasqueira, etc.)
- ✅ Painel de comunicados e avisos
- ✅ Administração geral do condomínio
